# Home Services App 🏠

Welcome to the Home Services App (Seepage repair, tiling, grouting, and welding). 
This repository contains both the React Native mobile application and the Next.js web application dashboard.

## Live Deployment

🌍 **Web Application (Vercel):** [https://home-services-app-seven.vercel.app/](https://home-services-app-seven.vercel.app/)

## Repository Structure

- `/homeserv-web` - The Next.js frontend application (Customer and Worker flows).
- `/src` - The React Native mobile application root logic.
- `AGENTS.md` & `CLAUDE.md` - Core AI architecture and instruction documentation.
